### Избранные проекты

**Qlearning_agent_hw**  
Реализация Q-learning агента с нуля. Сравнил две стратегии исследования окружения (ε-greedy и softmax), провёл анализ влияния гиперпараметров на стабильность и скорость обучения, добавил визуализацию кривых вознаграждения.  
→ [github.com/iljas277/Qlearning_agent_hw](https://github.com/iljas277/Qlearning_agent_hw)

**VoiceClassifier**  
Проект по классификации пола говорящего по аудиозаписи. Использовал mel-спектрограммы в качестве признаков и свёрточные нейронные сети.  
→ [github.com/iljas277/VoiceClassifier](https://github.com/iljas277/VoiceClassifier)

**mlflow-githubAuth**  
Полноценное развёртывание MLflow Tracking Server. Реализовал две версии: с Nginx в качестве reverse proxy и с аутентификацией через GitHub OAuth.  
→ [github.com/iljas277/mlflow-githubAuth](https://github.com/iljas277/mlflow-githubAuth)

**embadder**  
Утилита для генерации плотных эмбеддингов текстов и быстрого семантического поиска с помощью FAISS.  
→ [github.com/iljas277/embadder](https://github.com/iljas277/embadder)
